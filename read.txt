
Загрузить гит можно тут: http://git-scm.com/downloads

Устанавливайте гит с настройками по умолчанию.

Генерация пары ssh ключей:

     $ ssh-keygen -t rsa -C "lenkopv@gmail.com"

Публичный ключ (id_rsa) нужно оправить владельцу репозитория для получения прав работы. Или загрузить в настройки профиля в bitbucket/github/gitlab

Настройка имени пользователя и емейла:

$ git config --global user.name "Pavel Lenko"
$ git config --global user.email "lenkopv@gmail.com"
-----------------------------------------
Создать репозиторий на github и склонировать его локально
$ cd /d/git/myfirst

$ git clone git@github.com:plenko/myfirst.git
-----------------------------------------
добавление файла
$ git add <file> - добавление в индекс

$ git status - статус

$ git commit -m "comment" - закомитить

$ git log - история коммитов

$ git push - добавление данных локального репозитория на удаленный

$ git pull добавление данных удаленного репозитория на локальный
-----------------------------------------
$ git gui& - вызов GUI

$ gitk - история коммитов
.gitignore - скрыть файлы для git
-----------------------------------------
$ git checkout --<file>  - отменить изменения в одном modified file  (не добавленные в индекс)
$ git checkout . - отменить изменения во всех  modified file   (не добавленные в индекс)
$ git clean -xdf - удалить новые файлы

$ git reset --<file> - вернуть в unstaged (из добавленных в индекс в недобавленные)

$ git commit -ammend - дописать  commit in local branch (обновить предыдущий commit)
$ git reset HEAD^^  (HEAD~2) удалить  commit in local branch
$ git reset --soft   - from commit to index
$ git reset --mixed  - from commit to unstaged
$ git reset --hard   - delete commit
$ git revert sha1 (sha1 взять из git log) delete from remote repo
--------------------
$ git checkout -b 'feature' - Создать ветку и перейти в нее
